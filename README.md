# Autonomous-Trash-Collector-Boat-using-Raspberry-Pi-and-MobileNetSSD
A Raspberry-Pi based autonomous trash collector robot that laverages the use of MobileNetSSD for detection of trash and collects them, useful for small ponds and still waters.

# File Structure
Put Entire Object_Detection_Files folder onto the Raspberry pi and run object_ident_updated (2).py
coconames contains the name of the objects that can be detected using this pipeline. Inside update object_ident_updated.py, you can also select the names of the objects that you want to detect/omit.
This model/method was taken from: https://core-electronics.com.au/guides/object-identify-raspberry-pi/
